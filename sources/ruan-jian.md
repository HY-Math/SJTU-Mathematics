# 软件

## $$\LaTeX$$

数学系学生的必备技能。

### 使用方法

* 线上：[Overleaf](https://www.overleaf.com/)、[SJTU版Overleaf](https://latex.sjtu.edu.cn/project)，不用安装，点击即用，但可能不稳定。
* 本地：下载[TeXlive](https://tug.org/texlive/)，编辑器推荐[TeXstudio](https://texstudio.sourceforge.net)。

### 教程

* 《LaTeX入门》by刘海洋
